# 👁️‍🗨️ All Eyes
Herramienta de OSINT todo en uno, impulsada por machine learning.
Capaz de recolectar, analizar, clasificar e inferir informacion faltante a partir de patrones de aprendisaje automáticos, esta herramienta OSINT permite descubrir gran cantidad de información en la red.
Con multiples secciones que pronto se iran expandiendo:

- People Search Engine --> Se encarga de la busqueda de usuarios, informacion de edad, estudios, redes sociales, etc. Y armar un perfil.
- General Seach --> Se encarga de busquedas generales usando motores comunes para identificar probables correlaciones o patrones.
- IMINT / GEOINT --> Inteligencia de fuentes fotográficas y busqueda inversa para aproximar ubicación, fecha u metadata.
- Dark Web Search --> Implementacion de busqueda mediante la red TOR para analizar fuentes no publicas o brechas de seguridad.
- Network General Lookup --> Busqueda general, filtracion y obtención de informacion referente a redes.
